# Lifegame ( Game of Life ) on Launchpad with Node.js

[Novation Launchpad mini](http://us.novationmusic.com/launch/launchpad-mini)

# usage

Connect Novation Launchpad mini to your PC, then run below

```shell
npm install
node index.js
```
